# Fork 说明

本仓库为以下上游项目的维护分支，当前以 `Chat Capsule` 名称独立发布。该分支为非官方维护线，与上游维护者无隶属或背书关系：

- 上游项目：`WeFlow`
- 上游仓库：https://github.com/hicccc77/WeFlow
- 上游许可证：`CC BY-NC-SA 4.0`

## 本分支的主要变化

本分支定位为 **macOS 优先、完全本地、只读优先** 的桌面应用，并强化安全默认值。主要变化包括：

- 恢复 Electron 窗口隔离并保持 `webSecurity: true`
- 移除渲染层暴露的原始 SQL 与写数据库能力
- 加固 preload / IPC / 本地 HTTP API 边界
- 本地媒体访问改为受控协议
- 增加启动诊断、发布校验与安全相关文档
- 发布与打包流程独立，面向 `magic8ytes/Chat-Capsule` 发布线

## 署名与再分发

由于上游项目使用 `CC BY-NC-SA 4.0` 许可证，本分支继续保留署名，并在相同许可证体系与非商业约束下发布衍生内容。

本仓库使用新的产品标识（`Chat Capsule`）以避免与上游品牌或发布权产生混淆。再次分发前，请先阅读 `LICENSE` 并确保你的用途符合署名、非商业、相同方式共享等约束。
